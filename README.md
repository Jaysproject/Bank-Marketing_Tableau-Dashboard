# Bank Marketing Tableau Project

## Overview
This project analyses customer deposit behaviour using Tableau dashboards.

## Key Analysis
- Age-based conversion trends (U-shaped pattern)
- Job-wise deposit likelihood
- Impact of call duration on success rate
- Campaign strategy dashboard

## Key Insights
- Customers aged 18–24 and 65+ show the highest conversion rates
- Longer call durations strongly correlate with higher success
- Students and retirees are the most responsive segments
